# Luna's Tattoo Studio - Blackink.tattoo

Professional tattoo studio website for Luna at OZ Tattoo, Hollywood Blvd, Los Angeles, CA.

## Features
- Responsive design
- Online booking form
- Portfolio gallery
- SMS opt-in/opt-out functionality
- Privacy Policy and Terms of Service

## Technologies
- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- Feather Icons
